# 📝 To-Do List App

A simple To-Do List application built with **HTML, CSS, and JavaScript**.  
This app lets users add, delete, and mark tasks as completed.

## 🚀 Features
- Add new tasks  
- Mark tasks as complete  
- Delete tasks  
- Responsive and clean design
